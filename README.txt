Random, a great plugin designed to insert random contents, posts and other types, into your website. Enjoy its shortcodes!

== Description ==

With Random plugin you can insert into your WordPress website one or more random contents coming from your posts.
The source contents can be pages, posts or custom post types.
You can display information in many different ways:
* A list of post titles
* One or more full contents or excerpts
* Raw URLs to posts permalink

== Installation ==

From your WordPress dashboard

1. Visit ‘Plugins > Add New’
2. Search for ‘Random’
3. Activate the plugin from your Plugins page.

From WordPress.org

1. Download Random zip file
2. Unzip it
3. Upload the unzipped directory to your ‘/wp-content/plugins/’ directory
4. Activate the plugin from your Plugins page.

== Screenshots ==

1. Select how many post and with which types you want to show. Select also the post format.
2. For each post select the fields to show from title, content, excerpt or simple permalink.
3. Get the shortcode in one click; copy and paste it wherever you want in your webpage to show the random contents. Get also an instant demonstration of your shortcode result.
